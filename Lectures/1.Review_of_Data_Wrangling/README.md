## What is Model Ready (tidy) Data?

* [Tidy Data Paper by Hadley Wickham](https://vita.had.co.nz/papers/tidy-data.pdf)

## Key Concepts

### Column properties
- have names
- contain variables

### Row properties
- contain observations
- are exchangable
- order does not matter

### General Vocabulary

- Id variables
- Measured variables
- Filtering
- Transforming
- Aggregating

- Keys
- Indices
